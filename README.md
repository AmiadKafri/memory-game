# משחק הזכרון בעברית 🎮

משחק זכרון אינטראקטיבי עם מילים בעברית לילדים, המשלב טרגול הגייה וחיזוק זיכרון.

## פיצ'רים

✨ **בחירת אות**: בחר כל אות מהא"ב העברי (א-ת)

📍 **בחירת מיקום**: בחר היכן תופעה לאות במילה:
- תחילת המילה
- אמצע המילה
- סוף המילה

📊 **רמות קושי**:
- קל: 4 זוגות
- בינוני: 6 זוגות
- קשה: 8 זוגות

🎯 **משחק דינמי**: המשחק נוצר דינמית בהתאם לבחירות שלך

📈 **ניקוד**: עקוב אחרי מספר נסיונות וזמן משחק

## התקנה

### דרישות מקדימות
- Node.js (גרסה 16 ומעלה)
- npm או yarn

### צעדים

1. **שכפל את הפרויקט** (או עבור לתיקייה שלו)
   ```bash
   cd "c:\Projects\memory game"
   ```

2. **התקן תלויות**
   ```bash
   npm install
   ```

3. **הרץ את השרת המקומי**
   ```bash
   npm run dev
   ```

   התוכנה תיפתח אוטומטית ב- `http://localhost:3000`

## שימוש

1. **בחר אות** - בחר אות כלשהי מהא"ב
2. **בחר מיקום** - בחר היכן תופעה לאות במילה
3. **בחר רמה** - בחר רמת קושי
4. **שחק** - הפוך כרטיסים וחפש התאמות זהות

## מבנה הפרויקט

```
src/
├── components/
│   ├── LetterSelector.jsx      # בחירת אות
│   ├── PositionSelector.jsx    # בחירת מיקום
│   ├── LevelSelector.jsx       # בחירת רמה
│   ├── GameBoard.jsx           # לוח המשחק
│   └── Card.jsx                # כרטיס משחק בודד
├── styles/
│   ├── Card.css
│   ├── GameBoard.css
│   └── Selector.css
├── data/
│   └── words.json              # מאגר מילים

├── main.jsx                    # נקודת כניסה
└── index.css                   # סגנונות גלובליים
```
